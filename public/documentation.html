API documentation for the myFlix app

This API helps serve the myFlix app clients with the required backend resources

Request URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET none A JSON object containing data of all the movies
Get data of single movie by title /movies/[title] GET none A JSON object containing data of a single movie
Get description of a genre by genre name /movies/genre/[genreName]/ GET none A string containing data of the genre
Get data about a director by director's name /movies/director/[name]/ GET none A JSON object containing director's name,bio, birth and death years
Get a single user by username /users/[username] GET none A JSON object containing data of single user
Allow new users to register /users POST A JSON object holding data about the new user structured like (fields with * required):
{
Username*:"BrianMukukeba",
Password*:"brian2000";
Email*:"brian@yahoo.com"
Birthdate:new date(Year-Month-Day)
FavouriteMovies:[MovieIDs]
} 		A JSON object holding data about the new user structured like:
{
Username:string,
Password:string;
Email:string
Birthdate:date
FavouriteMovies:[strings]
_Id:ObjectId("string") }
Update user info by username /users/[username] PUT A JSON object holding data about the updated user info structured like :
{
Username:"BrianMukukeba",
Password:"brian2000";
Email:"brian@yahoo.com"
Birthdate:new date(Year-Month-Day)
} 		A JSON object holding data about the updated user info structured like:
{
Username:string,
Password:string;
Email:string
Birthdate:date
FavouriteMovies:[strings]
_Id:ObjectId("string") }
Add a movie to list of favourite movies /users/[username]/movie/[MovieId] Put none A message indicating that movie has been succesfully added to list of favourites
Allow users remove a movie from their list of favourites /users/[username]/favMovies/[removeMovie]
DELETE none A message indicating that movie has been succesfully removed from list of favourites
Deregister or delete user account /users/[username] DELETE none A message indicating delete status
