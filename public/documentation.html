API documentation for the myFlix app

This API helps serve the myFlix app clients with the required backend resources

Request URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET none A JSON object containing data of all the movies
Get data of single movie /movies/[title] GET none A JSON object containing data of a single movie
Get description of a genre by genre name /movies/genre/[name]/ GET none A string containing data of a single movie
Get data about a director by director's name /movies/director/[name]/ GET none A JSON object containing director's name and day of birth
Allow new users to register /users POST A JSON object holding data about the new user structured like:
{
username:"Brian Mukukeba",
password:"xyz!";
} 		A JSON object holding data about the new user structured like:
{
username:"Brian Mukeba",
password:"xyz!";
id:1
}
Allow users update their password /users/[username]/[password] PUT none A message indicating that user sussesfully updated his/her password
Allow users add a movie to their list of favourite movies /users/[username]/favMovies/[favMovies] Put none A message indicating that movie has been succesfully added to list of favourites
Allow users remove a movie from their list of favourites /users/[username]/favMovies/[removeMovie]
DELETE none A message indicating that movie has been succesfully removed from list of favourites
Allow users to deregister /users/[id] DELETE none A message indicating that user has been succesfully deregistered
